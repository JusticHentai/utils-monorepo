# command-exists

commandExists 是一个用于判断系统命令是否存在的异步函数，支持 Windows 和 Linux/macOS 系统。

## 使用示例

<CodeOrSourceMdx language="typescript">
  {normalDemo}
</CodeOrSourceMdx>

## 参数介绍

### command

- 类型：`string`
- 描述：要检测的命令名称

### 返回值

- 类型：`Promise<boolean>`
- 描述：命令是否存在
