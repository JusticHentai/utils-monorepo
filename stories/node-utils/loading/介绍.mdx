# Loading

Loading 是一个用于在终端显示 loading 动画的类，基于 ora 库封装。

## 使用示例

<CodeOrSourceMdx language="typescript">
  {normalDemo}
</CodeOrSourceMdx>

## 方法介绍

### start

- 类型：`(text?: string) => void`
- 描述：开始显示 loading 动画
- 参数：`text` - 可选，loading 文案，默认为 `'开始加载'`

### succeed

- 类型：`(text?: string) => void`
- 描述：显示成功状态并停止 loading
- 参数：`text` - 可选，成功文案，默认为 `'加载成功'`

### fail

- 类型：`(text?: string) => void`
- 描述：显示失败状态并停止 loading
- 参数：`text` - 可选，失败文案，默认为 `'加载失败'`

### stop

- 类型：`() => void`
- 描述：停止 loading 动画
