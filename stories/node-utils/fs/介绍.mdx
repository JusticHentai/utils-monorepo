# fs

fs 模块提供了一系列文件系统操作工具，基于 fs-extra 封装，带有 loading 提示和日志输出。

## cd - 切换目录

<CodeOrSourceMdx language="typescript">
  {cdDemo}
</CodeOrSourceMdx>

### 参数

- `dirname`: `string` - 目标目录路径，传入 `-` 可切换回原始目录
- 返回值: `Promise<boolean>` - 是否切换成功

## copy - 复制文件/目录

<CodeOrSourceMdx language="typescript">
  {copyDemo}
</CodeOrSourceMdx>

### 参数

- `sourcePath`: `string` - 源路径
- `targetPath`: `string` - 目标路径
- 返回值: `Promise<boolean>` - 是否复制成功

> 注意：自动过滤 node_modules 和 dist 目录
