## 配置详解

### 项目基本信息

#### `name: "@justichentai/utils-monorepo"`
- **作用**：项目名称
- **说明**：使用 scoped package 命名（`@justichentai/` 前缀）

#### `version: "1.0.0"`
- **作用**：项目版本号

#### `description: "持续集成工具链 monorepo"`
- **作用**：项目描述
- **说明**：说明这是一个持续集成工具链的 monorepo 项目

#### `type: "module"`
- **作用**：指定项目使用 ES 模块
- **说明**：
  - 启用 ES6 模块语法（`import/export`）
  - 所有 `.js` 文件默认被视为 ES 模块
  - 无需使用 `.mjs` 扩展名

### 脚本命令

#### `build: "pnpm run -r build"`
- **作用**：构建所有包
- **说明**：
  - `-r`：递归执行，在所有 workspace 包中运行
  - 构建 monorepo 中的所有包

#### `cli: "tsx ./cli/src/index.ts"`
- **作用**：运行 CLI 工具
- **说明**：使用 `tsx` 直接运行 TypeScript 文件

#### `cli:init: "pnpm run -F @justichentai/cli link:global"`
- **作用**：全局安装 CLI 工具
- **说明**：
  - `-F`：过滤，只运行指定包的命令
  - `link:global`：将 CLI 工具链接到全局

#### `md: "storybook dev -p 6006"`
- **作用**：启动 Storybook 开发服务器
- **说明**：
  - 在端口 6006 上运行 Storybook
  - 用于查看和开发组件文档

#### `md:build: "storybook build"`
- **作用**：构建 Storybook 静态站点
- **说明**：生成用于部署的静态文件
