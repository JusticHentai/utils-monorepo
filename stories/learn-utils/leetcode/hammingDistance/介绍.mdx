# 汉明距离 (LeetCode 461)

## 题目描述

两个整数之间的汉明距离指的是这两个数字对应二进制位不同的位置的数目。给你两个整数 `x` 和 `y`，计算并返回它们之间的汉明距离。

## 核心原理

**位运算**：
- 将两个数转换为二进制字符串
- 对齐后逐位比较不同的位数

## 解题思路

1. 将两个数转换为二进制字符串
2. 用 0 补齐到相同长度
3. 逐位比较，统计不同的位数

## 复杂度分析

- **时间复杂度**：O(log(max(x, y)))
- **空间复杂度**：O(log(max(x, y)))
