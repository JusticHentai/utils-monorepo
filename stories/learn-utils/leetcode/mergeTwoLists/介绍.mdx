# mergeTwoLists 合并两个有序链表

[LeetCode 21. 合并两个有序链表](https://leetcode.cn/problems/merge-two-sorted-lists/) 是链表的基础题。

## 题目描述

将两个升序链表合并为一个新的升序链表并返回。

## 解题思路

### 迭代法

1. 创建哑节点作为新链表头
2. 比较两链表当前节点，较小的接入新链表
3. 处理剩余节点
