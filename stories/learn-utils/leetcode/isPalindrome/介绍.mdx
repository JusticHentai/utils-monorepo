# 回文链表 (LeetCode 234)

## 题目描述

给你一个单链表的头节点 `head`，请你判断该链表是否为回文链表。如果是，返回 `true`；否则，返回 `false`。

## 核心原理

**快慢指针 + 反转链表**：
1. 使用快慢指针找到链表中点
2. 反转后半部分链表
3. 比较前半部分和反转后的后半部分

## 解题思路

1. 快指针每次走 2 步，慢指针每次走 1 步，找到中点
2. 将链表从中点断开
3. 反转后半部分链表
4. 逐一比较两个链表的节点值

## 复杂度分析

- **时间复杂度**：O(n)
- **空间复杂度**：O(1)
