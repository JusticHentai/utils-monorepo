# generateParenthesis 括号生成

[LeetCode 22. 括号生成](https://leetcode.cn/problems/generate-parentheses/) 是回溯算法的经典题目。

## 题目描述

给出 n 代表生成括号的对数，请你生成所有可能的并且有效的括号组合。

## 解题思路

### DFS + 剪枝

1. 左括号数量小于 n 时可以添加左括号
2. 右括号数量小于左括号数量时可以添加右括号
3. 字符串长度达到 2n 时记录结果
