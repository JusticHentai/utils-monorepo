# rotate 旋转图像

[LeetCode 48. 旋转图像](https://leetcode.cn/problems/rotate-image/) 是矩阵操作的经典题目。

## 题目描述

给定一个 n × n 的二维矩阵 matrix 表示一个图像，请你将图像顺时针旋转 90 度。

## 解题思路

### 原地旋转

1. 每次旋转四个位置的元素
2. 只需处理 1/4 的元素
3. 使用临时变量保存被覆盖的值
