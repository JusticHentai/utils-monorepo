# productExceptSelf 除自身以外数组的乘积

[LeetCode 238. 除自身以外数组的乘积](https://leetcode.cn/problems/product-of-array-except-self/) 是前缀积的经典题目。

## 题目描述

给你一个整数数组 nums，返回数组 answer，其中 answer[i] 等于 nums 中除 nums[i] 之外其余各元素的乘积。不能使用除法。

## 解题思路

### 左右乘积

1. 先计算每个位置左边所有元素的乘积
2. 再从右往左乘以右边所有元素的乘积
