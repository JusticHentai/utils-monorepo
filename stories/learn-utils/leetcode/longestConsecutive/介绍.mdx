# 最长连续序列 (LeetCode 128)

## 题目描述

给定一个未排序的整数数组 `nums`，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。请你设计并实现时间复杂度为 O(n) 的算法解决此问题。

## 核心原理

**哈希表**：
- 使用 Set 存储所有数字
- 只从连续序列的起点开始计数

## 解题思路

1. 将所有数字存入 Set
2. 遍历 Set，对于每个数字 `num`
3. 如果 `num - 1` 不在 Set 中，说明 `num` 是序列起点
4. 从起点开始向后计数连续数字
5. 更新最长序列长度

## 复杂度分析

- **时间复杂度**：O(n)
- **空间复杂度**：O(n)
