# maxArea 盛最多水的容器

[LeetCode 11. 盛最多水的容器](https://leetcode.cn/problems/container-with-most-water/) 是双指针的经典题目。

## 题目描述

给定一个长度为 n 的整数数组 height。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i])。找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

## 解题思路

### 双指针

1. 左右指针从两端开始
2. 计算当前面积，更新最大值
3. 移动较短的那个指针

## 为什么移动较短的指针？

移动较长的指针不会使面积增大，因为面积由较短的边决定。
