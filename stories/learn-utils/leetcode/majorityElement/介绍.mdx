# majorityElement 多数元素

[LeetCode 169. 多数元素](https://leetcode.cn/problems/majority-element/) 是哈希表/摩尔投票的经典题目。

## 题目描述

给定一个大小为 n 的数组 nums，返回其中的多数元素。多数元素是指在数组中出现次数大于 ⌊n/2⌋ 的元素。

## 解题思路

### 哈希表计数

1. 统计每个元素出现次数
2. 找到出现次数超过 n/2 的元素
