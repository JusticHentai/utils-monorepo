# numIslands 岛屿数量

[LeetCode 200. 岛屿数量](https://leetcode.cn/problems/number-of-islands/) 是 DFS/BFS 的经典题目。

## 题目描述

给你一个由 '1'（陆地）和 '0'（水）组成的二维网格，请你计算网格中岛屿的数量。

## 解题思路

### DFS 感染法

1. 遍历网格，遇到 '1' 则岛屿数量 +1
2. 从该位置开始 DFS，将连通的 '1' 都标记为 '0'
3. 继续遍历直到结束
