# uniquePaths 不同路径

[LeetCode 62. 不同路径](https://leetcode.cn/problems/unique-paths/) 是动态规划的经典题目。

## 题目描述

一个机器人位于一个 m x n 网格的左上角，机器人每次只能向下或者向右移动一步。问总共有多少条不同的路径？

## 解题思路

### 动态规划

1. **状态定义**：`dp[i][j]` 表示到达 (i,j) 的路径数
2. **状态转移**：`dp[i][j] = dp[i-1][j] + dp[i][j-1]`
3. **边界条件**：第一行和第一列都是 1
