# subsets 子集

[LeetCode 78. 子集](https://leetcode.cn/problems/subsets/) 是回溯算法的经典题目。

## 题目描述

给你一个整数数组 nums，数组中的元素互不相同。返回该数组所有可能的子集。

## 解题思路

### DFS 选或不选

1. 对于每个元素，有两种选择：选或不选
2. 递归处理所有元素
3. 到达末尾时记录当前组合
