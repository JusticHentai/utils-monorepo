# addTwoNumbers 两数相加

[LeetCode 2. 两数相加](https://leetcode.cn/problems/add-two-numbers/) 是链表操作的经典题目。

## 题目描述

给你两个非空的链表，表示两个非负的整数。它们每位数字都是按照逆序的方式存储的，将两个数相加，并以相同形式返回一个表示和的链表。

## 解题思路

1. **模拟加法**：从低位到高位逐位相加
2. **进位处理**：使用 carry 记录进位
3. **链表构建**：边计算边构建结果链表
