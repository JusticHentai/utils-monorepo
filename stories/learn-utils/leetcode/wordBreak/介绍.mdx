# wordBreak 单词拆分

[LeetCode 139. 单词拆分](https://leetcode.cn/problems/word-break/) 是动态规划/记忆化搜索的经典题目。

## 题目描述

给你一个字符串 s 和一个字符串列表 wordDict 作为字典。请你判断是否可以利用字典中出现的单词拼接出 s。

## 解题思路

### DFS + 记忆化

1. 从位置 0 开始，尝试匹配字典中的单词
2. 匹配成功则递归处理剩余部分
3. 使用 memo 数组记录已计算的结果
