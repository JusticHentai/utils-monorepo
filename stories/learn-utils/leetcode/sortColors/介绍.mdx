# sortColors 颜色分类

[LeetCode 75. 颜色分类](https://leetcode.cn/problems/sort-colors/) 是双指针的经典题目，又称荷兰国旗问题。

## 题目描述

给定一个包含红色、白色和蓝色（0、1、2）的数组 nums，原地对它们进行排序。

## 解题思路

### 双指针

1. p0 指向下一个 0 应该放置的位置
2. p1 指向下一个 1 应该放置的位置
3. 遍历数组，遇到 0 或 1 进行交换
