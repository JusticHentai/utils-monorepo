# 相交链表 (LeetCode 160)

## 题目描述

给你两个单链表的头节点 `headA` 和 `headB`，请你找出并返回两个单链表相交的起始节点。如果两个链表不存在相交节点，返回 `null`。

## 核心原理

**双指针**：
- 两个指针分别从两个链表头部出发
- 当一个指针到达末尾时，跳转到另一个链表头部
- 两指针相遇时即为相交节点

## 解题思路

1. 指针 pA 从 headA 出发，指针 pB 从 headB 出发
2. pA 到达末尾后跳转到 headB，pB 到达末尾后跳转到 headA
3. 两指针相遇时即为相交节点（或同时为 null 表示不相交）

## 复杂度分析

- **时间复杂度**：O(m + n)
- **空间复杂度**：O(1)
