# 在排序数组中查找元素的第一个和最后一个位置 (LeetCode 34)

## 题目描述

给你一个按照非递减顺序排列的整数数组 `nums`，和一个目标值 `target`。请你找出给定目标值在数组中的开始位置和结束位置。如果数组中不存在目标值 `target`，返回 `[-1, -1]`。

## 核心原理

**两次二分查找**：
- 第一次找左边界（第一个等于 target 的位置）
- 第二次找右边界（最后一个等于 target 的位置）

## 解题思路

1. 二分查找左边界：找第一个 >= target 的位置
2. 二分查找右边界：找第一个 > target 的位置 - 1
3. 验证找到的位置是否有效

## 复杂度分析

- **时间复杂度**：O(log n)
- **空间复杂度**：O(1)
