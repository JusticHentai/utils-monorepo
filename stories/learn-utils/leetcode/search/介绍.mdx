# search 搜索旋转排序数组

[LeetCode 33. 搜索旋转排序数组](https://leetcode.cn/problems/search-in-rotated-sorted-array/) 是二分查找的变体题。

## 题目描述

整数数组 nums 按升序排列，数组中的值互不相同。在传递给函数之前，nums 在预先未知的某个下标 k 上进行了旋转。给你旋转后的数组 nums 和一个整数 target，如果 nums 中存在这个目标值，则返回它的下标，否则返回 -1。

## 解题思路

### 二分查找

1. 找到中点，判断哪半部分是有序的
2. 判断 target 是否在有序部分
3. 根据判断结果缩小搜索范围
