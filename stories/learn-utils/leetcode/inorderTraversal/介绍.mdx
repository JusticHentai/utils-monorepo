# 二叉树的中序遍历 (LeetCode 94)

## 题目描述

给定一个二叉树的根节点 `root`，返回它的中序遍历。

## 核心原理

**递归遍历**：
- 中序遍历顺序：左子树 → 根节点 → 右子树
- 递归访问每个节点

## 解题思路

1. 如果节点为空，返回
2. 递归遍历左子树
3. 访问当前节点
4. 递归遍历右子树

## 复杂度分析

- **时间复杂度**：O(n)
- **空间复杂度**：O(n)，递归栈空间
