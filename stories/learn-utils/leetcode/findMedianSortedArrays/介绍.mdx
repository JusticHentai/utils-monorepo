# 寻找两个正序数组的中位数 (LeetCode 4)

## 题目描述

给定两个大小分别为 `m` 和 `n` 的正序（从小到大）数组 `nums1` 和 `nums2`。请你找出并返回这两个正序数组的中位数。算法的时间复杂度应该为 O(log(m+n))。

## 核心原理

**二分查找**：
- 在较短数组上进行二分查找
- 找到一个分割点，使得左半部分的最大值小于等于右半部分的最小值

## 解题思路

1. 确保 nums1 是较短的数组
2. 在 nums1 上二分查找分割点 i，nums2 的分割点 j 自动确定
3. 调整分割点直到满足：`nums1[i-1] <= nums2[j]` 且 `nums2[j-1] <= nums1[i]`
4. 根据总长度奇偶性计算中位数

## 复杂度分析

- **时间复杂度**：O(log(min(m, n)))
- **空间复杂度**：O(1)
