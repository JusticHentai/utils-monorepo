# merge 合并区间

[LeetCode 56. 合并区间](https://leetcode.cn/problems/merge-intervals/) 是区间问题的经典题目。

## 题目描述

以数组 intervals 表示若干个区间的集合，其中单个区间为 intervals[i] = [starti, endi]。请你合并所有重叠的区间，并返回一个不重叠的区间数组。

## 解题思路

### 排序 + 合并

1. 按区间起点排序
2. 遍历区间，判断是否与前一个重叠
3. 重叠则合并，否则加入结果
