# 最大正方形 (LeetCode 221)

## 题目描述

在一个由 `'0'` 和 `'1'` 组成的二维矩阵内，找到只包含 `'1'` 的最大正方形，并返回其面积。

## 核心原理

**动态规划**：
- `dp[i][j]` 表示以 `(i, j)` 为右下角的最大正方形边长
- 状态转移：`dp[i][j] = min(dp[i-1][j-1], dp[i][j-1], dp[i-1][j]) + 1`

## 解题思路

1. 创建 dp 数组
2. 遍历矩阵，如果当前位置是 '1'
3. `dp[i][j] = min(左上, 上, 左) + 1`
4. 记录最大边长，返回面积

## 复杂度分析

- **时间复杂度**：O(m * n)
- **空间复杂度**：O(m * n)
