# maxDepth 二叉树的最大深度

[LeetCode 104. 二叉树的最大深度](https://leetcode.cn/problems/maximum-depth-of-binary-tree/) 是递归的入门题。

## 题目描述

给定一个二叉树，找出其最大深度。

## 解题思路

### DFS 递归

1. 递归计算左子树深度
2. 递归计算右子树深度
3. 返回 `1 + max(左深度, 右深度)`
