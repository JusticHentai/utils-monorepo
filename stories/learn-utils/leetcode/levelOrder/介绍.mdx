# levelOrder 二叉树的层序遍历

[LeetCode 102. 二叉树的层序遍历](https://leetcode.cn/problems/binary-tree-level-order-traversal/) 是 BFS 的经典应用。

## 题目描述

给你二叉树的根节点 root，返回其节点值的层序遍历。

## 解题思路

### BFS + 队列

1. 使用队列存储每层节点
2. 每次处理一层的所有节点
3. 将子节点加入队列
