# 从前序与中序遍历序列构造二叉树 (LeetCode 105)

## 题目描述

给定两个整数数组 `preorder` 和 `inorder`，其中 `preorder` 是二叉树的先序遍历，`inorder` 是同一棵树的中序遍历，请构造二叉树并返回其根节点。

## 核心原理

1. **前序遍历特点**：根节点 → 左子树 → 右子树
2. **中序遍历特点**：左子树 → 根节点 → 右子树
3. **递归构建**：前序的第一个元素是根节点，在中序中找到根节点位置，左边是左子树，右边是右子树

## 解题思路

1. 前序遍历的第一个元素是根节点
2. 在中序遍历中找到根节点的位置 `mid`
3. 中序遍历中 `mid` 左边的元素属于左子树，右边属于右子树
4. 递归构建左右子树

## 复杂度分析

- **时间复杂度**：O(n)，n 为节点数
- **空间复杂度**：O(n)，递归栈空间
