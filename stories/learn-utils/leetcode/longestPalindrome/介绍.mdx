# longestPalindrome 最长回文子串

[LeetCode 5. 最长回文子串](https://leetcode.cn/problems/longest-palindromic-substring/) 是动态规划的经典题目。

## 题目描述

给你一个字符串 s，找到 s 中最长的回文子串。

## 解题思路

### 动态规划

1. **状态定义**：`dp[i][j]` 表示 s[i..j] 是否为回文
2. **状态转移**：`dp[i][j] = s[i] === s[j] && (j-i < 3 || dp[i+1][j-1])`
3. 从短到长遍历所有子串
