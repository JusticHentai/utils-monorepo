# permute 全排列

[LeetCode 46. 全排列](https://leetcode.cn/problems/permutations/) 是回溯算法的经典题目。

## 题目描述

给定一个不含重复数字的数组 nums，返回其所有可能的全排列。

## 解题思路

### 回溯法

1. 每次从剩余元素中选择一个
2. 递归处理剩余元素
3. 回溯时恢复状态
