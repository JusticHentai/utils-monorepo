# 正则表达式匹配 (LeetCode 10)

## 题目描述

给你一个字符串 `s` 和一个字符规律 `p`，请你来实现一个支持 `.` 和 `*` 的正则表达式匹配。
- `.` 匹配任意单个字符
- `*` 匹配零个或多个前面的那一个元素

## 核心原理

**动态规划**：
- `dp[i][j]` 表示 `s` 的前 `i` 个字符与 `p` 的前 `j` 个字符是否匹配
- 根据当前字符和 `*` 的情况进行状态转移

## 解题思路

1. 初始化：`dp[0][0] = true`，处理 `p` 以 `*` 开头的情况
2. 如果 `p[j-1]` 是普通字符或 `.`，检查是否匹配
3. 如果 `p[j-1]` 是 `*`，考虑匹配 0 次或多次

## 复杂度分析

- **时间复杂度**：O(m * n)
- **空间复杂度**：O(m * n)
