# findKthLargest 数组中的第K个最大元素

[LeetCode 215. 数组中的第K个最大元素](https://leetcode.cn/problems/kth-largest-element-in-an-array/) 是堆排序的经典应用。

## 题目描述

给定整数数组 nums 和整数 k，请返回数组中第 k 个最大的元素。

## 解题思路

### 堆排序

1. 构建大顶堆
2. 依次取出堆顶元素 k-1 次
3. 第 k 次堆顶即为答案
