# coinChange 零钱兑换

[LeetCode 322. 零钱兑换](https://leetcode.cn/problems/coin-change/) 是动态规划的经典题目。

## 题目描述

给你一个整数数组 coins 表示不同面额的硬币，以及一个整数 amount 表示总金额。计算凑成总金额所需的最少硬币个数。

## 解题思路

### 动态规划

1. **状态定义**：`dp[i]` 表示凑成金额 i 所需的最少硬币数
2. **状态转移**：`dp[i] = min(dp[i], dp[i - coin] + 1)`
3. **初始状态**：`dp[0] = 0`，其他为 Infinity
