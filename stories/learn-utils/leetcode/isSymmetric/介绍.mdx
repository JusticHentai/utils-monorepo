# 对称二叉树 (LeetCode 101)

## 题目描述

给你一个二叉树的根节点 `root`，检查它是否轴对称。

## 核心原理

**递归比较**：
- 对称二叉树满足：左子树的左节点 = 右子树的右节点，左子树的右节点 = 右子树的左节点
- 递归比较两棵子树

## 解题思路

1. 定义递归函数比较两个节点
2. 如果两个节点都为空，返回 true
3. 如果只有一个为空，返回 false
4. 如果值不相等，返回 false
5. 递归比较：左节点的左子树与右节点的右子树，左节点的右子树与右节点的左子树

## 复杂度分析

- **时间复杂度**：O(n)
- **空间复杂度**：O(n)，递归栈空间
