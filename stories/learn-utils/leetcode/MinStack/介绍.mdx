# MinStack 最小栈

[LeetCode 155. 最小栈](https://leetcode.cn/problems/min-stack/) 设计一个支持 O(1) 获取最小值的栈。

## 题目描述

设计一个支持 `push`、`pop`、`top` 操作，并能在常数时间内检索到最小元素的栈。

## 解题思路

### 辅助栈

1. **数据栈**：存储所有元素
2. **最小栈**：同步存储当前最小值
3. **同步操作**：push/pop 时两个栈同步操作
