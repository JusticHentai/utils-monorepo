# maxSubArray 最大子数组和

[LeetCode 53. 最大子数组和](https://leetcode.cn/problems/maximum-subarray/) 是动态规划的入门经典题。

## 题目描述

给你一个整数数组 `nums`，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

## 解题思路

### 动态规划（Kadane 算法）

1. **状态定义**：`pre` 表示以当前元素结尾的最大子数组和
2. **状态转移**：`pre = max(pre + num, num)`
3. **结果更新**：每次更新全局最大值

### 核心思想

- 如果前面的和是负数，那么加上当前数只会更小
- 所以直接从当前数重新开始计算
