# throttleByRaf

throttleByRaf 是一个基于 requestAnimationFrame 的节流函数，保证同一帧内只执行一次回调。

## 基础示例

<CodeOrSourceMdx language="typescript">
  {basicDemo}
</CodeOrSourceMdx>

## 参数介绍

### cb

- 类型：`(...params: any[]) => any`
- 描述：需要节流的回调函数

### 返回值

- 类型：`[Cb, () => void]`
- 描述：返回一个元组
  - 第一个元素：节流后的函数
  - 第二个元素：取消函数，用于在组件卸载时取消未执行完的帧，防止内存泄漏

## 特点

- 基于 requestAnimationFrame 实现，保证同一帧内不会重复执行
- 提供 cancel 函数，便于在组件卸载时清理
- 适合用于高频触发的场景，如滚动、拖拽等
