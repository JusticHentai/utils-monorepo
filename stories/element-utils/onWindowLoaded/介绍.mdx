# onWindowLoaded

在窗口加载完成时执行回调。

## 详细介绍

`onWindowLoaded` 用于监听 `window` 的 `load` 事件。如果页面尚未加载完成（`document.readyState !== 'complete'`），则注册事件监听；否则延迟一个宏任务后立即执行回调。

延迟执行的目的是确保浏览器在 `load` 事件触发后完成所有同步的性能条目（如 `PerformanceNavigationTiming`）写入，避免回调中读取到不完整的性能数据。

常用于性能监控中采集导航性能指标。

## 具体实现原理

1. **文件职责**：`index.ts` 判断 `document.readyState` 决定是监听还是延迟执行
2. **核心流程**：readyState 不为 `complete` 时通过 `createEventListener` 注册 load 监听，否则通过 `await sleep(0)` 延迟后执行回调
3. **关键技术点**：使用 `sleep(0)` 延迟确保浏览器性能条目写入完成后再执行回调，避免读取不完整的性能数据
