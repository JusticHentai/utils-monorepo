# isPureElement

判断 React 子元素是否是纯元素（string 或 number）。

## 参数介绍

### el

- 类型：`ReactNode`
- 描述：要判断的 React 子元素

### 返回值

- 类型：`el is string | number`
- 描述：如果元素是 string 或 number 类型返回 true，否则返回 false
