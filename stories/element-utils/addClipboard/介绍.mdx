# addClipboard

将文本复制到剪贴板的异步函数。

## 详细介绍

addClipboard 是一个剪贴板操作函数，核心作用是：**将指定文本复制到系统剪贴板**。

**涉及的知识点**：
- `navigator.clipboard.writeText()`：现代浏览器剪贴板 API
- Promise：异步操作包装
- 元组返回：`[success, error]` 形式便于错误处理

**常用场景**：
- 复制链接：一键复制分享链接
- 复制代码：代码块的复制功能
- 复制文本：复制表格单元格、ID 等内容
- 邀请码：复制邀请码或优惠券

**注意事项**：
- 需要 HTTPS 或 localhost 环境
- 需要用户交互触发（点击事件等）
- 需要用户授权剪贴板权限

### 返回值

- 类型：`Promise<[boolean, any]>`
- 描述：元组形式，`[success, error]`
  - 成功：`[true, undefined]`
  - 失败：`[false, Error]`

## 具体实现原理

### 文件结构

- `index.ts` - 唯一源文件，包含 addClipboard 函数实现

### 核心流程

1. 调用 `navigator.clipboard.writeText(text)`
2. 成功则返回 `[true, undefined]`
3. 失败则捕获错误，返回 `[false, error]`

### 关键技术点

- 使用 Clipboard API：`navigator.clipboard.writeText(text)`
- 返回元组而非 reject，调用方无需 try/catch
- Clipboard API 是异步的，返回 Promise
