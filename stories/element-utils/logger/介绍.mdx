# logger

带颜色标签的日志输出工具。

## 详细介绍

logger 是一个美化日志输出的工具，核心作用是：**在控制台输出带有彩色标签的日志，便于区分不同级别的信息**。

**涉及的知识点**：
- console.log 的 CSS 样式：`%c` 占位符
- 日志级别：info、debug、warn、error
- 颜色区分：不同级别使用不同颜色

**常用场景**：
- 开发调试：快速定位不同类型的日志
- 模块标识：区分不同模块的输出
- 错误追踪：醒目标记错误信息
- 性能调试：标记性能相关的日志

**优势**：
- 颜色醒目，一眼区分日志级别
- API 简单，与 console.log 用法一致
- 支持任意数量的参数

## 具体实现原理

### 文件结构

- `index.ts` - 唯一源文件，包含 logger 对象和各级别方法

### 核心流程

1. 定义不同级别的颜色映射
2. 各方法调用 `console.log('%c[LEVEL]', 'color: xxx', ...args)`
3. 使用 `%c` 占位符应用 CSS 样式

### 关键技术点

- 使用 `console.log('%c text', 'css style')` 为文字添加样式
- 样式包含：color、background、padding、border-radius 等
- 第一个参数是带标签的字符串，后续参数为实际日志内容
