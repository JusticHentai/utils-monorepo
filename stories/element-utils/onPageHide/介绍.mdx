# onPageHide

监听 pagehide 事件。

## 详细介绍

`onPageHide` 监听 `window` 上的 `pagehide` 事件，在页面即将被隐藏或卸载时触发。回调参数为 `PageTransitionEvent`，其中 `persisted` 属性表示页面是否可能进入 BFCache。

推荐使用 `pagehide` 替代 `unload` 事件，因为 `unload` 会阻止 BFCache。常用于数据持久化和最终的数据上报。

## 返回值

返回 `() => void` 清理函数。

## 具体实现原理

1. **文件职责**：`index.ts` 封装 `createEventListener(window, 'pagehide', callback)`
2. **关键技术点**：`persisted` 为 `true` 时页面进入 BFCache，后续可通过 `pageshow` 事件恢复
