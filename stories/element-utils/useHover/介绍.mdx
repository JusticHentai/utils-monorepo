# useHover

监听 DOM 元素是否有鼠标悬停的 Hook。

## 详细介绍

useHover 通过监听 mouseenter 和 mouseleave 事件来追踪元素的悬停状态，并支持进入、离开和状态变化的回调函数。

### 使用场景

- 悬停显示提示信息
- 悬停改变元素样式
- 悬停触发动画效果
- 下拉菜单的显示/隐藏

## 具体实现原理

### 核心流程

1. 使用 useState 管理悬停状态
2. 使用 useEventListener 监听 mouseenter 和 mouseleave 事件
3. 事件触发时更新状态并调用相应的回调函数
