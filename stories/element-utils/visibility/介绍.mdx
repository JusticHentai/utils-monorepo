# visibility

visibility 是一个用于监听页面可见性变化（visibilitychange）事件的函数。

## 基础示例

<CodeOrSourceMdx language="typescript">
  {basicDemo}
</CodeOrSourceMdx>

## 参数介绍

### options

- 类型：`Options`
- 描述：配置对象

#### options.visible

- 类型：`() => void`
- 描述：页面变为可见时触发的回调函数

#### options.hidden

- 类型：`() => void`
- 描述：页面变为隐藏时触发的回调函数

### 返回值

- 类型：`() => void`
- 描述：移除事件监听的清理函数

## 特点

- 基于 visibilitychange 事件
- 分别处理可见和隐藏状态
- 返回清理函数，便于组件卸载时移除监听
- 适用于暂停/恢复动画、节省资源等场景
