# isElementChildren

判断某个元素是否是另一个元素的子元素。

  {basicDemo}
</CodeOrSourceMdx>

## 非子元素判断

  {notChildDemo}
</CodeOrSourceMdx>

## 参数介绍

### element

- 类型：`Element`
- 描述：父元素

### children

- 类型：`Element`
- 描述：要判断的子元素

### 返回值

- 类型：`boolean`
- 描述：如果 children 是 element 的子元素返回 true，否则返回 false
