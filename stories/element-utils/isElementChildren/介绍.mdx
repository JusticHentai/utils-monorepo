# isElementChildren

isElementChildren 是一个用于判断某个元素是否是另一个元素的子元素的函数。

## 基础示例

<CodeOrSourceMdx language="typescript">
  {basicDemo}
</CodeOrSourceMdx>

## 参数介绍

### element

- 类型：`Element`
- 描述：父元素

### children

- 类型：`Element`
- 描述：要判断的子元素

### 返回值

- 类型：`boolean`
- 描述：如果 children 是 element 的子元素，返回 true，否则返回 false

## 特点

- 递归遍历 DOM 树判断父子关系
- 处理了空值情况，避免报错
