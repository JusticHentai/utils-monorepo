# mergeQueryParams

将源 URL 的查询参数合并到目标 URL 中。

## 参数介绍

### sourceUrl

- 类型：`string`
- 描述：源 URL，其查询参数将被提取

### targetUrl

- 类型：`string`
- 描述：目标 URL，源 URL 的查询参数将被添加到此 URL

### 返回值

- 类型：`string`
- 描述：添加查询参数后的完整 URL
