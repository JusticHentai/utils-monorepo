# mergeQueryParams

mergeQueryParams 是一个用于将源 URL 的查询参数合并到目标 URL 中的函数。

## 基础示例

<CodeOrSourceMdx language="typescript">
  {basicDemo}
</CodeOrSourceMdx>

## 参数介绍

### sourceUrl

- 类型：`string`
- 描述：源 URL，其查询参数将被提取

### targetUrl

- 类型：`string`
- 描述：目标 URL，源 URL 的查询参数将被添加到此 URL

### 返回值

- 类型：`string`
- 描述：添加查询参数后的完整 URL

## 特点

- 使用 URLSearchParams 解析参数
- 使用 URL API 构建结果
- 同名参数会被覆盖
