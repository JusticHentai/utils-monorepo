# mergeChildren

mergeChildren 是一个用于合并 React 子元素中相邻纯元素（string/number）的函数。

## 源码

<CodeOrSourceMdx language="typescript">
  {sourceCode}
</CodeOrSourceMdx>

## 参数介绍

### children

- 类型：`ReactNode`
- 描述：React 子元素

### 返回值

- 类型：`ReactNode[]`
- 描述：合并后的子元素数组，相邻的纯文本会被合并，最终的字符串会被包裹在 span 中

## 特点

- 合并相邻的纯文本节点
- 最终的字符串类型会被包裹在 span 标签中
- 使用 React.Children 遍历子元素
- 适用于优化 React 渲染性能
