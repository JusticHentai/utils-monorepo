# onWindowResize

监听窗口尺寸变化。

## 详细介绍

`onWindowResize` 用于监听浏览器窗口尺寸变化，支持节流优化。

常用场景：
- 响应式布局调整
- 图表/Canvas 重绘
- 虚拟列表尺寸更新
- 移动端横竖屏切换

## 使用示例

```typescript
import onWindowResize from '@justichentai/element-utils/onWindowResize'

const stop = onWindowResize(({ width, height }) => {
  console.log('窗口尺寸:', width, 'x', height)
})

// 带节流
const stop2 = onWindowResize(
  ({ width, height }) => console.log(width, height),
  { throttle: 100 }
)
```
