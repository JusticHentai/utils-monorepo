# onElementResize

监听元素尺寸变化（带防抖）。使用 ResizeObserver 监听元素尺寸变化，支持防抖处理避免频繁触发。

## 详细介绍

`onElementResize` 是基于 `ResizeObserver` 封装的尺寸变化监听工具，内置防抖处理。

**涉及的知识点**：
- ResizeObserver API：浏览器原生的元素尺寸监听 API
- 防抖处理：避免高频回调影响性能
- borderBoxSize：获取元素的边框盒尺寸

**常用场景**：
- 响应式布局：元素尺寸变化时重新计算布局
- 图表重绘：容器尺寸变化时重绘图表
- 编辑器：调整编辑区域大小
- 可拖拽面板：监听面板尺寸变化

**优势**：
- 内置防抖，减少回调频率
- 比 window.resize 更精确
- 自动获取宽高信息

## 具体实现原理

### 文件结构

- `index.ts` - 主函数实现
- `interface.ts` - 类型定义

### 核心流程

1. 根据 delay 参数决定是否包装防抖
2. 调用 `createResizeObserver` 创建观察器
3. 将防抖后的回调传递给观察器

### 关键技术点

- 复用 `debounce` 工具函数实现防抖
- delay 为 0 时不进行防抖包装
- 回调参数包含 width 和 height
