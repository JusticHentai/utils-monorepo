# onPageResume

监听页面 resume 事件（Page Lifecycle API）。

## 详细介绍

`onPageResume` 监听 `document` 上的 `resume` 事件，当页面从冻结状态恢复时触发。与 `onPageFreeze` 对应，用于恢复冻结前暂停的操作。

## 具体实现原理

1. **文件职责**：`index.ts` 封装 `createEventListener(document, 'resume', callback)`
2. **关键技术点**：`resume` 是 Page Lifecycle API 的一部分，与 `freeze` 事件配对使用
