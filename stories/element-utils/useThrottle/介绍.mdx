# useThrottle

useThrottle 是一个 React Hook，用于在 React 组件中实现节流功能。

## 源码

  {sourceCode}

## 参数介绍

### Options

#### cb

- 类型：`<T extends any[]>(...args: T) => any`
- 描述：需要节流的回调函数

#### duration

- 类型：`number`
- 描述：节流时间间隔（毫秒）

### 返回值

- 类型：`<T extends any[]>(...args: T) => void`
- 描述：返回节流后的函数

## 特点

- 使用 useRef 维护 timer，保证在 React Fiber 中正确工作
- 立即执行回调，然后在 duration 时间内忽略后续调用
- 适用于 React 组件中的高频事件处理
