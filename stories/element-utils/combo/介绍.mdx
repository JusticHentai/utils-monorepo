# combo

combo 是一个用于实现连击效果的函数，常用于游戏或交互场景。

## 基础示例

<CodeOrSourceMdx language="typescript">
  {basicDemo}
</CodeOrSourceMdx>

## 参数介绍

### options

- 类型：`ComboOptions`
- 描述：配置对象

<table>
  <thead>
    <tr>
      <th>参数名</th>
      <th>类型</th>
      <th>描述</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>onCombo</td>
      <td><code>(combo?: number) =&gt; void</code></td>
      <td>连击回调函数，参数为当前连击次数</td>
    </tr>
    <tr>
      <td>onClose</td>
      <td><code>() =&gt; void</code></td>
      <td>连击结束时触发的回调</td>
    </tr>
    <tr>
      <td>duration</td>
      <td><code>number</code></td>
      <td>连击间隔时间（毫秒），超过此时间未触发则连击结束</td>
    </tr>
  </tbody>
</table>

### 返回值

- 类型：`() => void`
- 描述：返回触发连击的函数，每次调用会增加连击计数
