# onPageShow

监听 pageshow 事件。

## 详细介绍

`onPageShow` 监听 `window` 上的 `pageshow` 事件。该事件在页面首次加载和从 BFCache 恢复时都会触发。回调参数为 `PageTransitionEvent`，`persisted` 为 `true` 表示页面从 BFCache 恢复。

常用于 Web Vitals 监控中检测 BFCache 恢复后重新上报性能指标。

## 具体实现原理

1. **文件职责**：`index.ts` 封装 `createEventListener(window, 'pageshow', callback)`
2. **关键技术点**：`persisted` 属性区分首次加载和 BFCache 恢复
