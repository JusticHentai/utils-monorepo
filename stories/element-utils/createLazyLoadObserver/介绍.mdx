# createLazyLoadObserver

创建懒加载观察器。批量监听多个元素的可见性，当元素进入视口时执行加载函数。

## 详细介绍

`createLazyLoadObserver` 是用于批量懒加载的工具函数，每个元素只会加载一次，加载后自动停止观察该元素。

**涉及的知识点**：
- IntersectionObserver API：浏览器原生的元素可见性检测
- WeakSet：使用弱引用集合记录已加载元素
- 批量观察：一个观察器监听多个元素

**常用场景**：
- 图片懒加载：大量图片只在进入视口时加载
- 列表项懒渲染：长列表中的项目按需渲染
- 广告位懒加载：广告内容延迟加载
- 组件懒初始化：复杂组件延迟初始化

**优势**：
- 批量处理，一个观察器管理多个元素
- 自动去重，每个元素只加载一次
- 支持提前加载（rootMargin）
- 统一清理函数

## 具体实现原理

### 文件结构

- `index.ts` - 主函数实现
- `interface.ts` - 类型定义

### 核心流程

1. 创建 `WeakSet` 记录已加载的元素
2. 使用 `createIntersectionObserver` 观察所有元素
3. 元素进入视口时检查是否已加载
4. 未加载则执行加载函数，标记为已加载，并取消对该元素的观察

### 关键技术点

- 使用 `WeakSet` 避免内存泄漏
- 调用 `observer.unobserve(target)` 单独停止已加载元素的观察
- 统一的 `stop` 函数用于清理所有观察
