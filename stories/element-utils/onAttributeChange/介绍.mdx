# onAttributeChange

监听元素属性变化。使用 MutationObserver 监听指定元素的属性变化，可以获取属性名、旧值和新值。

## 详细介绍

`onAttributeChange` 是基于 `MutationObserver` 封装的属性变化监听工具。

**涉及的知识点**：
- MutationObserver API：浏览器原生的 DOM 变化监听 API
- 属性变化监听：监听元素的 class、style、data-* 等属性变化
- attributeFilter：可选择性地只监听特定属性

**常用场景**：
- 表单状态监控：监听输入框的 disabled、readonly 等属性变化
- 样式变化追踪：监听 class 或 style 属性的动态变化
- 数据属性监听：监听 data-* 属性的变化用于组件通信
- 调试工具：追踪 DOM 属性的修改记录

**优势**：
- 精确获取旧值和新值
- 支持过滤特定属性
- 比轮询检测更高效

## 具体实现原理

### 文件结构

- `index.ts` - 主函数实现
- `interface.ts` - 类型定义

### 核心流程

1. 调用 `createMutationObserver` 创建观察器
2. 配置 `attributes: true` 和 `attributeOldValue: true`
3. 回调中过滤 `mutation.type === 'attributes'` 的变化
4. 从 mutation 中提取属性名、旧值，并获取新值

### 关键技术点

- 使用 `attributeOldValue: true` 获取属性旧值
- 使用 `attributeFilter` 选项优化性能
- 新值需要通过 `element.getAttribute()` 获取
