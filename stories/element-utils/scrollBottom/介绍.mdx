# scrollBottom

scrollBottom 是一个用于监听元素滚动到底部事件的函数。

## 基础示例

<CodeOrSourceMdx language="typescript">
  {basicDemo}
</CodeOrSourceMdx>

## 参数介绍

### element

- 类型：`HTMLElement`
- 描述：需要监听的 DOM 元素

### cb

- 类型：`(...params: any[]) => any`
- 描述：滚动到底部时触发的回调函数

### threshold

- 类型：`number`
- 默认值：`0`
- 描述：滚动到底部的阈值（像素），默认 0 表示严格底部

### 返回值

- 类型：`() => void`
- 描述：移除事件监听的清理函数，用于防止内存泄漏

## 特点

- 支持自定义阈值，可在接近底部时提前触发
- 返回清理函数，便于组件卸载时移除监听
- 适用于无限滚动、加载更多等场景
