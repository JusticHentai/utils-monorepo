# onPageFreeze

监听页面 freeze 事件（Page Lifecycle API）。

## 详细介绍

`onPageFreeze` 监听 `document` 上的 `freeze` 事件，该事件属于 Page Lifecycle API，当浏览器决定冻结页面（如长时间后台标签页）时触发。冻结的页面不会执行定时器和回调，是浏览器节省资源的一种机制。

常用于在页面被冻结前保存状态或清理资源。

## 具体实现原理

1. **文件职责**：`index.ts` 封装 `createEventListener(document, 'freeze', callback)`
2. **关键技术点**：`freeze` 事件是 Page Lifecycle API 的一部分，仅在支持该 API 的浏览器中可用（主要是 Chrome）
