# loadFont

异步加载字体并添加到 document.fonts 中。

## 基础示例

<CodeOrSourceMdx language="typescript">
  {basicDemo}
</CodeOrSourceMdx>

## 参数介绍

### url

- 类型：`string`
- 描述：字体文件地址

### name

- 类型：`string`
- 描述：字体名称，加载后可通过该名称在 CSS 中使用

### 返回值

- 类型：`Promise<[FontFace | undefined, Error | undefined]>`
- 描述：返回一个元组
  - 第一个元素：加载成功时返回 FontFace 对象，失败时为 undefined
  - 第二个元素：加载失败时返回 Error，成功时为 undefined
