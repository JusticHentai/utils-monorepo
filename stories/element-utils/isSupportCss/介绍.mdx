# isSupportCss

判断浏览器是否支持某些 CSS 属性。

## 参数介绍

### css

- 类型：`Record<string, string>`
- 描述：要检测的 CSS 属性对象，键为属性名，值为属性值

### 返回值

- 类型：`boolean`
- 描述：如果所有 CSS 属性都支持返回 true，否则返回 false
