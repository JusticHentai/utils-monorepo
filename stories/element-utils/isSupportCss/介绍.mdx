# isSupportCss

isSupportCss 是一个用于判断浏览器是否支持某些 CSS 属性的函数。

## 基础示例

<CodeOrSourceMdx language="typescript">
  {basicDemo}
</CodeOrSourceMdx>

## 参数介绍

### css

- 类型：`Record<string, string>`
- 描述：要检测的 CSS 属性对象，键为属性名，值为属性值

### 返回值

- 类型：`boolean`
- 描述：如果所有 CSS 属性都支持，返回 true，否则返回 false

## 特点

- 优先使用 CSS.supports API
- 降级使用创建元素检测的方式
- 支持同时检测多个 CSS 属性
