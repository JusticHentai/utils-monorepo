# useBoolean

管理 boolean 状态的 Hook。

## 详细介绍

useBoolean 是 useToggle 的语义化封装，专门用于管理布尔值状态。它提供了更直观的 API，如 `setTrue`、`setFalse`，让代码更具可读性。

### 使用场景

- 控制模态框显示/隐藏
- 控制加载状态
- 控制开关状态
- 控制展开/折叠状态

## 具体实现原理

useBoolean 基于 useToggle 实现，将其返回的通用 toggle 方法封装成更语义化的布尔操作方法。

### 核心流程

1. 调用 useToggle，传入布尔化后的 defaultValue
2. 使用 useMemo 创建 setTrue、setFalse 等方法
3. 返回状态和操作方法
