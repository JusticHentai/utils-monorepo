# result

解析对象指定路径的值。

## 详细介绍

`result` 解析对象指定路径的值，如果值是函数则调用并返回结果，值为 undefined 时返回默认值。

**常用场景**：
- 安全地获取嵌套属性
- 自动调用方法属性
- 带默认值的属性访问

## 路径值解析

使用 `result` 解析对象路径上的值，支持自动调用函数和默认值。

## 具体实现原理

1. **文件职责**: `index.ts` 实现带函数调用的路径解析
2. **核心流程**: 解析路径 → 逐级访问 → 如果值是函数则调用 → undefined 时返回默认值
3. **关键技术点**: 路径解析、函数类型检测与自动调用、默认值回退
4. **数据流向**: 对象 + 路径 → 逐级访问 → 函数调用/取值 → 返回结果
