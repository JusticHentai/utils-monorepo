# defaultTo

检查 value 是否为 null/undefined/NaN，如果是则返回默认值。

## 详细介绍

`defaultTo` 提供安全的默认值回退机制。当值为 `null`、`undefined` 或 `NaN` 时返回指定的默认值。相比 `||` 运算符，它不会将 `0` 或 `''` 视为假值。

## 具体实现原理

1. **文件职责**: `defaultTo/index.ts` 导出 `defaultTo` 函数
2. **核心流程**: 检查 value 是否为 null、undefined 或 NaN，是则返回 defaultValue
3. **关键技术点**: `Number.isNaN` 判断 NaN，`== null` 同时检查 null 和 undefined
4. **数据流向**: value → 检查是否为空值/NaN → 返回 value 或 defaultValue
