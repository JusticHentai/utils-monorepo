# formatTime

formatTime 是一个用于格式化时间戳为数字数组格式的函数，返回 [时, 分, 秒] 的数组。

  {hourDemo}

## 参数介绍

### ts

- 类型：`number`
- 描述：时间戳（秒级）

### 返回值

- 类型：`number[]`
- 描述：数字数组格式的时间 [时, 分, 秒]，例如 126秒 → [0, 0, 0, 2, 0, 6] 表示 00:02:06
