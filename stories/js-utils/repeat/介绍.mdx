# repeat

重复字符串 n 次。

## 详细介绍

`repeat` 将字符串重复指定次数。

**常用场景**：
- 生成重复模式
- 字符串填充
- 分隔线生成

## 具体实现原理

1. **文件职责**: `index.ts` 实现字符串重复
2. **核心流程**: 将字符串重复 n 次拼接
3. **关键技术点**: 处理边界情况（0次、负数、非整数）
4. **数据流向**: 字符串 + 次数 → 重复拼接 → 返回结果
