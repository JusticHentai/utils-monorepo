# find

返回第一个满足谓词函数的元素，同时提供 `findLast` 从右查找。

## 详细介绍

`find` 遍历集合，返回第一个使谓词函数返回 true 的元素。`findLast` 从集合末尾开始查找。找不到则返回 undefined。

## 具体实现原理

1. **文件职责**: `find/index.ts` 导出 find（default）和 findLast（named）
2. **核心流程**: 遍历集合（正向或逆向），返回第一个满足谓词的元素
3. **关键技术点**: 短路返回、fromIndex 支持
4. **数据流向**: 集合 → 遍历 → 谓词匹配 → 返回匹配元素或 undefined
