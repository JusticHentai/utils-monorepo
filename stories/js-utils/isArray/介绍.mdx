# isArray

isArray 是一个用于检测值是否为数组的函数。

## 参数介绍

### obj

- 类型：`any`
- 描述：要检测的值

### 返回值

- 类型：`obj is any[]`
- 描述：如果值是数组则返回 `true`，否则返回 `false`
