# isArray

检测值是否为数组。

## 详细介绍

使用 `Object.prototype.toString.call()` 检测值是否为数组类型。对数组返回 `true`，对其他所有值（包括类数组对象）返回 `false`。

### 返回值
- 类型：`boolean`
- 描述：如果是数组返回 true

## 具体实现原理
1. **文件职责**: `isArray/index.ts` 导出 isArray
2. **核心流程**: 使用 `Object.prototype.toString.call(obj)` 判断
3. **关键技术点**: toString 标签比较 `[object Array]`
4. **数据流向**: value → toString → 比较标签 → 返回布尔值
