# sortedIndex

使用二分查找确定插入位置。

## 详细介绍

`sortedIndex` 系列函数使用二分查找确定值在已排序数组中的插入位置。包含 By、Of、Last 等变体。

**常用场景**：
- 有序数组的高效插入
- 二分查找定位
- 维护排序数组

### 返回值

- 类型：`number`
- 描述：值应插入的索引位置

## 具体实现原理

1. **文件职责**: `index.ts` 提供 sortedIndex 系列六个函数
2. **核心流程**: 二分查找 → 比较中间值 → 缩小范围 → 返回插入位置
3. **关键技术点**: 二分查找算法，By 变体先通过迭代器转换再比较
4. **数据流向**: 已排序数组 + 值 → 二分查找 → 返回索引
