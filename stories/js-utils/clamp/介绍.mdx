# clamp

将数值限制在指定范围内。

## 详细介绍

`clamp` 将数值约束在 `[min, max]` 区间内。值在范围内时原样返回，超出范围时返回对应边界值。支持只限制最小值或只限制最大值。当 `min > max` 时会自动交换。

**常用场景**：
- 滑块/进度条范围限制
- 游戏中角色坐标/血量约束
- 动画进度值 0-1 限制
- 表单数字输入范围校验

## 基础范围限制

范围内的值不变，超出范围的值被限制到边界。

## 具体实现原理

1. **文件职责**: `clamp/index.ts` 导出 clamp 函数
2. **核心流程**: 判断 min/max 是否同时存在 → 同时存在时自动处理 min > max → 使用 Math.min/Math.max 限制
3. **关键技术点**: `Math.min`、`Math.max` 组合实现区间限制
4. **数据流向**: 输入 {value, min, max} → 计算有效边界 → 返回限制后的值
