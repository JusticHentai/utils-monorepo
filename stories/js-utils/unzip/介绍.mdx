# unzip

将分组数组重新按索引分组。

## 详细介绍

`unzip` 是 zip 的逆操作，将分组数组按原始索引重新分组。`unzipWith` 支持自定义组合逻辑。

**常用场景**：
- 分离组合数据
- 矩阵转置
- 数据重组

## 具体实现原理

1. **文件职责**: `index.ts` 提供 unzip 和 unzipWith
2. **核心流程**: 确定最长子数组长度 → 按索引位置重新分组
3. **关键技术点**: 矩阵转置思想，unzipWith 对每组应用组合函数
4. **数据流向**: 分组数组 → 按索引重组 → 返回新分组
