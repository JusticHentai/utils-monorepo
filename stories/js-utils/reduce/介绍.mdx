# reduce

遍历集合，逐步累积返回值。

## 详细介绍

`reduce` 对集合元素进行累积操作，支持数组和对象。`reduceRight` 从右到左遍历。

**常用场景**：
- 数组求和/求积
- 数据聚合
- 对象转换

## 具体实现原理

1. **文件职责**: `index.ts` 提供 reduce 和 reduceRight
2. **核心流程**: 初始化累积器 → 遍历集合 → 每次将前一步结果传入迭代器 → 返回最终值
3. **关键技术点**: 支持数组和对象，reduceRight 使用反向遍历
4. **数据流向**: 集合 + 初始值 → 逐步累积 → 返回最终结果
