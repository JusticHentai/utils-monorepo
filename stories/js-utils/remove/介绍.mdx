# remove

移除满足条件的数组元素（原地修改）。

## 详细介绍

`remove` 移除数组中所有满足谓词函数的元素，直接修改原数组，返回被移除的元素。

**常用场景**：
- 按条件批量删除元素
- 数据过滤（需要保留被删除项）
- 列表项管理

## 具体实现原理

1. **文件职责**: `index.ts` 实现条件元素移除
2. **核心流程**: 遍历数组 → 执行谓词 → 匹配则从原数组移除并收集
3. **关键技术点**: 原地修改数组，反向遍历避免索引偏移
4. **数据流向**: 数组 + 谓词 → 遍历判断 → 移除匹配项 → 返回被移除数组
