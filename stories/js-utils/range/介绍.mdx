# range

创建一个包含指定范围数字的数组。

## 详细介绍

`range` 函数用于生成数字序列数组，类似 Python 的 range 函数。

**包含的函数**：
- `range` - 升序生成
- `rangeRight` - 降序生成

**常用场景**：
- 循环指定次数
- 生成索引数组
- 创建分页页码
