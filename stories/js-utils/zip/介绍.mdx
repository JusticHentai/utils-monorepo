# zip

将多个数组按索引分组。

## 详细介绍

`zip` 将多个数组按索引分组。还提供 `zipObject`（创建键值对象）、`zipObjectDeep`（支持路径）、`zipWith`（自定义组合）。

**常用场景**：
- 合并对应位置的数据
- 从两个数组创建对象
- 矩阵转置

### 返回值

- 类型：`T[][]` / `Record<K, V>` / `R[]`
- 描述：分组后的数组或对象

## 具体实现原理

1. **文件职责**: `index.ts` 提供 zip、zipObject、zipObjectDeep、zipWith
2. **核心流程**: 找到最长数组长度 → 按索引位置收集各数组元素
3. **关键技术点**: zipObject 将键数组和值数组组合为对象，zipWith 对每组应用组合函数
4. **数据流向**: 多个数组 → 按索引分组 → 返回分组结果
