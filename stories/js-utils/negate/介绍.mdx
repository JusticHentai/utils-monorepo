# negate

创建取反的谓词函数。

## 详细介绍

`negate` 创建一个函数，返回原谓词函数结果的逻辑否定。

**常用场景**：
- 快速创建反向过滤条件
- 函数式编程中的逻辑取反
- 避免编写重复的否定逻辑
