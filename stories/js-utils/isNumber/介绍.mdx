# isNumber

判断值是否为数字（排除 NaN）。

## 详细介绍

使用 `Object.prototype.toString` 检测值是否为 Number 类型，并排除 NaN。注意：此实现与 lodash 不同，lodash 的 `isNumber(NaN)` 返回 `true`。

### 返回值
- 类型：`boolean`
- 描述：如果是数字（不含 NaN）返回 true

## 具体实现原理
1. **文件职责**: `isNumber/index.ts` 导出 isNumber
2. **核心流程**: toString 标签检查 + NaN 排除（`obj === obj`）
3. **关键技术点**: NaN 是唯一不等于自身的值
4. **数据流向**: value → toString → NaN check → 返回布尔值
