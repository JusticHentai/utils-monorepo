# lerp

线性插值函数，用于在两个值之间按比例取值。

## 详细介绍

lerp（Linear Interpolation，线性插值）是图形学和动画中最基础的插值算法，核心作用是：**根据比例 n，在 a 和 b 之间取一个中间值**。

**涉及的知识点**：
- 线性插值公式：`(1 - n) * a + n * b`
- 参数化方程：n 表示从 a 到 b 的"进度"

**常用场景**：
- 动画过渡：计算动画过程中的中间状态
- 颜色渐变：RGB 值的平滑过渡
- 缓动函数：配合 easing 曲线实现加速/减速
- 相机移动：平滑跟随目标位置
- 数值平滑：将突变转为渐变

**公式解释**：
- `n = 0` 时，返回 a（起点）
- `n = 1` 时，返回 b（终点）
- `n = 0.5` 时，返回 a 和 b 的中点

**优势**：
- 计算简单高效
- 适用于任意数值类型
- 是更复杂插值算法的基础

## 具体实现原理

### 文件结构

- `index.ts` - 唯一源文件，包含 lerp 函数实现

### 核心流程

1. 接收起点 a、终点 b、比例 n
2. 计算 `(1 - n) * a + n * b`
3. 返回插值结果

### 关键技术点

- 实现：`(1 - n) * a + n * b`
- 等价形式：`a + n * (b - a)`（先算差值再偏移）
- n ∈ [0, 1] 时结果在 [a, b] 范围内
- n 可以超出 [0, 1]，实现外插（Extrapolation）
