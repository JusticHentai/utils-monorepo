# lerp

lerp 是一个用于线性插值的函数，在两点连线上选择一点代表想要的值。

## 参数介绍

### a

- 类型：`number`
- 描述：低点

### b

- 类型：`number`
- 描述：高点

### n

- 类型：`number`
- 描述：当前选点到低点的距离，范围 [0, 1]

### 返回值

- 类型：`number`
- 描述：插值结果
