# assign

将源对象的自有可枚举属性分配到目标对象。

## 详细介绍

`assign` 函数将一个或多个源对象的自有可枚举属性复制到目标对象。后续源对象的同名属性会覆盖前面的值。它类似于 `Object.assign`，但提供了更一致的行为。

同时提供了 `assignIn`（包含继承属性）、`assignWith`（支持自定义合并函数）等变体。

**常用场景**：
- 合并配置对象
- 浅拷贝对象
- 扩展默认选项

## 具体实现原理

1. **文件职责**: `assign/index.ts` 导出 assign、assignIn、assignWith、assignInWith、extend、extendWith
2. **核心流程**: 遍历源对象的属性，依次赋值到目标对象
3. **关键技术点**: `Object.keys`（自有属性）vs `for...in`（含继承属性）
4. **数据流向**: 目标对象 → 遍历每个源对象 → 复制属性 → 返回修改后的目标对象
