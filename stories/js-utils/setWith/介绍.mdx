# setWith

在对象指定路径上设置值。

## 详细介绍

`setWith` 类似于 `set`，但支持自定义路径上创建的中间值。

**常用场景**：
- 深层嵌套对象属性设置
- 自定义中间层类型（如用数组代替对象）
- 配置对象构建

## 具体实现原理

1. **文件职责**: `index.ts` 实现带自定义器的深层路径设置
2. **核心流程**: 解析路径 → 逐级创建/访问属性 → 最终设置值
3. **关键技术点**: customizer 控制中间层对象类型，路径解析
4. **数据流向**: 对象 + 路径 + 值 → 逐级创建路径 → 设置最终值 → 返回对象
