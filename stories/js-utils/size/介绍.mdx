# size

返回集合的长度。

## 详细介绍

`size` 返回集合的大小。支持数组（length）、对象（键数量）和字符串（长度）。

**常用场景**：
- 通用集合大小获取
- 替代手动判断类型再取长度
- 数据验证

## 具体实现原理

1. **文件职责**: `index.ts` 实现通用集合大小获取
2. **核心流程**: 判断类型 → 数组/字符串取 length → 对象取 Object.keys.length
3. **关键技术点**: 类型判断，null/undefined 处理
4. **数据流向**: 集合 → 类型判断 → 对应方式获取大小 → 返回数值
