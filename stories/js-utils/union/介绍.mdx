# union

创建多个数组的并集。

## 详细介绍

`union` 创建包含所有唯一值的并集数组。`unionBy` 支持迭代器，`unionWith` 支持比较器。

**常用场景**：
- 合并去重多个数组
- 数据集合并
- 标签/分类合并

## 具体实现原理

1. **文件职责**: `index.ts` 提供 union、unionBy、unionWith
2. **核心流程**: 扁平化所有数组 → 按规则去重
3. **关键技术点**: union 用 Set 去重，unionBy 先迭代器映射再去重，unionWith 用比较器
4. **数据流向**: 多个数组 → 扁平化 → 去重 → 返回结果数组
