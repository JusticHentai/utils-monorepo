# isUrl

检查字符串是否为合法的 URL。

## 详细介绍

`isUrl` 检查给定字符串是否为合法的 URL 格式。

**常用场景**：
- 表单 URL 输入验证
- 链接合法性检查
- 数据清洗

## URL 格式检测
