# timeout

timeout 是一个用于同时执行异步函数和超时函数的函数，异步函数执行成功则返回异步函数的结果，超时函数执行成功则返回 undefined。

## 参数介绍

### 泛型

- `T` - 异步函数返回值的类型

### fn

- 类型：`() => Promise<T>`
- 描述：需要执行的异步函数

### ms

- 类型：`number`
- 描述：超时时间（毫秒）

### 返回值

- 类型：`Promise<T | undefined>`
- 描述：返回最先完成的结果或超时返回 undefined
