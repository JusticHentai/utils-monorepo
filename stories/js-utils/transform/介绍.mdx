# transform

对集合进行迭代累积转换。

## 详细介绍

`transform` 类似于 reduce，对对象或数组进行迭代累积。迭代器返回 false 可提前终止。

**常用场景**：
- 对象结构转换
- 数组归并
- 带提前终止的累积操作

## 具体实现原理

1. **文件职责**: `index.ts` 实现带提前终止的迭代累积
2. **核心流程**: 初始化累积器 → 遍历集合 → 调用迭代器修改累积器 → 返回 false 则终止
3. **关键技术点**: 迭代器直接修改累积器（非函数式），false 提前终止
4. **数据流向**: 集合 + 累积器 → 遍历迭代 → 修改累积器 → 返回结果
