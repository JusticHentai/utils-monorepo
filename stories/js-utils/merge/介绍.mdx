# merge

递归合并源对象的属性到目标对象。

## 详细介绍

`merge` 函数用于深度合并多个对象，与 `Object.assign` 不同，它会递归合并嵌套对象。

**与 assign 的区别**：
- `assign`：浅合并，直接覆盖嵌套对象
- `merge`：深合并，递归合并嵌套对象

**常用场景**：
- 合并配置对象
- 深度更新嵌套状态
- 合并默认值和用户设置
