# random

random 是一个用于获取 min 和 max 之间的随机整数的函数。

## 使用示例

<CodeOrSourceMdx language="typescript">
  {normalDemo}
</CodeOrSourceMdx>

## 参数介绍

### min

- 类型：`number`
- 描述：最小值（不包含）

### max

- 类型：`number`
- 描述：最大值（不包含）

### 返回值

- 类型：`number`
- 描述：min 和 max 之间的随机整数
