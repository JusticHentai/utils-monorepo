# words

将字符串拆分成单词数组。

## 详细介绍

`words` 将字符串拆分为单词数组，支持 camelCase、PascalCase、snake_case、kebab-case 等格式。

**常用场景**：
- 字符串格式转换的基础操作
- 文本分词
- 变量名解析

## 字符串分词

使用 `words` 将各种格式的字符串拆分为单词数组。

## 具体实现原理

1. **文件职责**: `index.ts` 实现字符串分词
2. **核心流程**: 使用正则匹配单词边界 → 提取所有单词

4. **数据流向**: 字符串 → 正则匹配 → 提取单词 → 返回数组
