# mapKeys

创建一个对象，键经过迭代器处理。

## 详细介绍

`mapKeys` 创建新对象，键为迭代器处理后的值，值保持不变。`mapValues` 则处理值保持键不变。

**常用场景**：
- 对象键名转换（如 camelCase 转 snake_case）
- 添加键前缀/后缀
- 值的批量转换

### 返回值

- 描述：键或值经过转换的新对象

## 具体实现原理

1. **文件职责**: `index.ts` 提供 mapKeys 和 mapValues 两个函数
2. **核心流程**: 遍历对象属性 → 调用迭代器转换键/值 → 构建新对象
3. **关键技术点**: 保持另一半（键或值）不变，仅转换目标部分
4. **数据流向**: 输入对象 + 迭代器 → 遍历属性 → 转换键/值 → 返回新对象
