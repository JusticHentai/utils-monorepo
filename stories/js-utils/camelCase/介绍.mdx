# camelCase

将字符串转换为驼峰命名格式。

## 详细介绍

`camelCase` 将各种格式的字符串（空格分隔、连字符分隔、下划线分隔、PascalCase 等）统一转换为小驼峰命名（lowerCamelCase）格式。内部先用 `words` 函数拆分单词，再将首个单词全小写、后续单词首字母大写后拼接。

**常用场景**：
- CSS 属性名转 JS 属性名（`background-color` → `backgroundColor`）
- API 字段命名风格统一
- 代码风格转换工具

## 基础转换

## 具体实现原理

1. **文件职责**: `camelCase/index.ts` 导出 camelCase 函数，依赖 `words` 函数拆分单词
2. **核心流程**: 输入字符串 → words 拆分为单词数组 → 首个单词全小写，后续首字母大写 → 拼接
3. **关键技术点**: `words` 函数使用正则匹配各种单词边界
4. **数据流向**: 字符串 → words() 拆分 → map 转换大小写 → join 拼接
