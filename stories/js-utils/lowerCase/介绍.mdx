# lowerCase

字符串大小写转换工具集。

## 详细介绍

`lowerCase` 将字符串转换为以空格分隔的小写形式。还提供 `upperCase`、`startCase`、`toLower`、`toUpper`。

**常用场景**：
- 文本格式化和规范化
- 标题生成（startCase）
- 不区分大小写的字符串处理

## 具体实现原理

1. **文件职责**: `index.ts` 提供多种字符串大小写转换函数
2. **核心流程**: 使用正则分割单词 → 转换每个单词大小写 → 重新拼接
3. **关键技术点**: 单词分割正则（处理 camelCase、snake_case 等格式）
4. **数据流向**: 输入字符串 → 正则分割单词 → 大小写转换 → 拼接返回
