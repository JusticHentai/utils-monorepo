# forEach

遍历集合中每个元素并执行回调函数。同时提供 `forEachRight` 从右遍历。

## 详细介绍

`forEach` 遍历集合元素，对每个元素调用 iteratee 函数。支持数组和对象。回调中返回 `false` 可提前终止遍历。`forEachRight` 从最后一个元素开始遍历。别名 `each` 和 `eachRight`。

## 具体实现原理

1. **文件职责**: `forEach/index.ts` 导出 forEach、forEachRight、each、eachRight
2. **核心流程**: for 循环遍历，支持 return false 中断
3. **关键技术点**: 支持数组和对象的统一遍历接口
4. **数据流向**: 集合 → 遍历每个元素 → 调用 iteratee → 返回原集合
