# kebabCase

转换字符串为短横线命名格式（kebab-case）。

## 详细介绍

`kebabCase` 函数将字符串转换为短横线分隔的小写格式。

**常用场景**：
- CSS 类名生成
- URL slug 生成
- HTML 属性名转换
