# isFunction

isFunction 是一个用于判断值是否为函数的函数。

</CodeOrSourceMdx>

  {notFunctionDemo}
</CodeOrSourceMdx>

## 参数介绍

### obj

- 类型：`any`
- 描述：要检测的值

### 返回值

- 类型：`obj is (...args: any[]) => any`
- 描述：如果值是函数则返回 `true`，否则返回 `false`
