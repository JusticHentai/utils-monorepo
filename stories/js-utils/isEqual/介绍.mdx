# isEqual

深度比较两个值是否相等。

## 详细介绍

`isEqual` 函数用于深度比较两个值，支持比较对象、数组、Date、RegExp 等类型。

**支持的类型**：
- 原始类型（使用 SameValueZero 比较）
- 对象和数组（递归比较）
- Date 和 RegExp
- Map 和 Set
- Buffer 和 TypedArray

**与 === 的区别**：
- `===` 只比较引用
- `isEqual` 深度比较内容

**常用场景**：
- 状态变化检测
- 缓存键比较
- 测试断言
