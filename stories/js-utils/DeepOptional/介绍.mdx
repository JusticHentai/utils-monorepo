# DeepOptional

DeepOptional 是一个用于将嵌套对象中指定路径的属性变为可选的类型工具。

## 使用示例

<CodeOrSourceMdx language="typescript">
  {normalDemo}
</CodeOrSourceMdx>

## 类型定义

```typescript
type DeepOptional<T, K extends Array<string | number | symbol>>
```

## 泛型参数

### T

- 类型：`object`
- 描述：要处理的对象类型

### K

- 类型：`Array<string | number | symbol>`
- 描述：属性路径数组，指定要变为可选的嵌套属性路径
