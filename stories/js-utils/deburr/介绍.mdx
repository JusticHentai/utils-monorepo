# deburr

将拉丁增补字母和拉丁扩展字母转换为基本拉丁字母，移除组合音标。

## 详细介绍

`deburr` 将带有变音符号的字符（如 é、ü、ñ）转换为对应的基本拉丁字母（e、u、n）。一些特殊字符有特定映射（如 Æ→Ae、ß→ss、Þ→Th）。数学运算符（×、÷）不会被转换。

**常用场景**：
- 搜索文本标准化（忽略变音符号）
- URL slug 生成
- 跨语言文本比较
- 排序时忽略变音差异

## 具体实现原理

1. **文件职责**: `deburr/index.ts` 导出 deburr 函数
2. **核心流程**: 遍历每个字符 → 查找映射表 → 替换或保留 → NFD 标准化后去除组合音标
3. **关键技术点**: 字符映射表（拉丁增补 + 扩展A）、`String.prototype.normalize('NFD')`、正则去除 \u0300-\u036f 范围
4. **数据流向**: 输入字符串 → 逐字符映射替换 → NFD 标准化 → 去除组合音标 → 返回
