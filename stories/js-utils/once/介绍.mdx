# once

创建只能调用一次的函数。

## 详细介绍

`once` 创建一个只能调用一次的函数。后续调用返回首次调用的结果。

**常用场景**：
- 初始化操作（只执行一次）
- 单例模式
- 一次性事件处理

## 单次调用函数

使用 `once` 确保函数只执行一次，后续调用返回首次结果。

## 具体实现原理

1. **文件职责**: `index.ts` 实现单次调用限制
2. **核心流程**: 包装原函数 → 首次调用时执行并缓存结果 → 后续调用直接返回缓存
3. **关键技术点**: 闭包保存调用状态和结果，标志位控制
4. **数据流向**: 原函数 → 包装 → 首次调用执行缓存 → 后续直接返回
