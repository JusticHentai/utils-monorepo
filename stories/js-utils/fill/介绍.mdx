# fill

使用指定值填充数组从 start 到 end（不含）的元素。

## 详细介绍

`fill` 函数用于使用指定值填充数组的指定范围，会修改原数组。

**常用场景**：
- 初始化数组
- 重置数组元素
- 创建指定长度和默认值的数组

## 具体实现原理

使用原生 `Array.prototype.fill()` 方法实现，支持负数索引。
