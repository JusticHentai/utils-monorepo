# isError

检查值是否为 Error 对象。

## 详细介绍

`isError` 支持检测 `Error`、`TypeError`、`RangeError` 等内置错误类型，以及 `DOMException`。通过 `Object.prototype.toString` 标签和 `instanceof` 检测。

### 返回值
- 类型：`boolean`
- 描述：如果是 Error 返回 true

## 具体实现原理
1. **文件职责**: `isError/index.ts` 导出 isError
2. **核心流程**: toString 标签检查 + instanceof 检查
3. **关键技术点**: 支持 `[object Error]` 和 `[object DOMException]`
4. **数据流向**: value → toString 标签 → instanceof → 返回布尔值
