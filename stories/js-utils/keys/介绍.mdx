# keys

获取对象自有可枚举属性的键名。

## 详细介绍

`keys` 返回对象自有可枚举属性的键名数组。还提供 `keysIn`（包含继承属性）、`values`、`valuesIn`。

**常用场景**：
- 遍历对象属性
- 获取对象大小
- 对象属性列表展示

### 返回值

- 类型：`string[]`
- 描述：对象自有可枚举属性的键名数组

## 具体实现原理

1. **文件职责**: `index.ts` 提供 keys、keysIn、values、valuesIn 四个函数
2. **核心流程**: 使用 Object.keys/for...in 获取属性键名或值
3. **关键技术点**: 区分自有属性和继承属性，hasOwnProperty 过滤
4. **数据流向**: 输入对象 → 属性遍历 → 过滤/收集 → 返回数组
