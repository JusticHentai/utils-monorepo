# sortedUniq

对已排序数组去重。

## 详细介绍

`sortedUniq` 利用已排序特性，只比较相邻元素来高效去重。`sortedUniqBy` 支持迭代器。

**常用场景**：
- 排序后的数组去重（比 uniq 更高效）
- 大数据集的去重优化
- 维护有序唯一列表

## 具体实现原理

1. **文件职责**: `index.ts` 提供 sortedUniq 和 sortedUniqBy
2. **核心流程**: 遍历已排序数组 → 仅比较相邻元素 → 不同则保留
3. **关键技术点**: 利用有序性只需 O(n) 时间，sortedUniqBy 通过迭代器转换后比较
4. **数据流向**: 已排序数组 → 相邻比较 → 保留不同元素 → 返回新数组
