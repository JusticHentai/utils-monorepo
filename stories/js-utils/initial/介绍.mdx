## 具体实现原理

1. **文件职责**: `initial/index.ts` 导出 `initial` 函数
2. **核心流程**: 使用 slice(0, -1) 返回除最后一个元素外的所有元素
3. **关键技术点**: `Array.prototype.slice`
4. **数据流向**: 数组 → slice(0, -1) → 返回新数组
