# propertyOf

与 property 相反，创建一个函数，返回给定路径在指定对象上的值

## 详细介绍

`propertyOf` 与 `property` 的柯里化参数顺序相反。

**与 `property` 的区别**：

<table>
  <thead>
    <tr>
      <th>函数</th>
      <th>先固定</th>
      <th>返回的函数接收</th>
      <th>适用场景</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>property</code></td>
      <td>路径</td>
      <td>不同对象</td>
      <td>对多个对象提取同一属性，如 <code>objects.map(property('a.b'))</code></td>
    </tr>
    <tr>
      <td><code>propertyOf</code></td>
      <td>对象</td>
      <td>不同路径</td>
      <td>对同一对象提取多个属性，如 <code>paths.map(propertyOf(obj))</code></td>
    </tr>
  </tbody>
</table>

**常用场景**：
- 数据处理和转换
- 功能组合和增强
- 简化常见操作
