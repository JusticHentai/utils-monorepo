# functions

返回对象自身的函数属性名数组。

## 详细介绍

`functions` 返回对象自身可枚举属性中类型为函数的键名。`functionsIn` 包含继承的属性。

### 返回值
- 类型：`string[]`

## 具体实现原理
1. **文件职责**: `functions/index.ts` 导出 functions（default）和 functionsIn（named）
2. **核心流程**: 遍历对象属性，筛选 typeof 为 function 的键
3. **关键技术点**: Object.keys 获取自身属性，for...in 获取继承属性
4. **数据流向**: object → 遍历键 → typeof 检查 → 返回函数名数组
