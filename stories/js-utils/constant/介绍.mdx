# constant

创建一个返回常量值的函数。

## 详细介绍

`constant` 创建一个无论如何调用都返回相同值的函数。在函数式编程中常作为默认回调或占位函数使用，例如 `Array.from({ length: 3 }, constant(0))` 创建全零数组。

## 具体实现原理

1. **文件职责**: `constant/index.ts` 导出 `constant` 函数
2. **核心流程**: 通过闭包捕获传入的值，返回一个始终返回该值的函数
3. **关键技术点**: 闭包
4. **数据流向**: value → 闭包捕获 → 返回的函数始终返回 value
