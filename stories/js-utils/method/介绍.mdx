# method

创建调用指定路径方法的函数。

## 详细介绍

`method` 创建一个函数，该函数调用传入对象上指定路径的方法。`methodOf` 是反向版本，先绑定对象。

**常用场景**：
- 动态方法调用
- 函数式编程中的方法引用
- 配置驱动的对象操作

## 具体实现原理

1. **文件职责**: `index.ts` 提供 method 和 methodOf 两个函数
2. **核心流程**: method 接收路径和参数 → 返回闭包 → 闭包接收对象并调用路径方法
3. **关键技术点**: 路径解析、参数预设、method/methodOf 互为反向操作
4. **数据流向**: 路径 + 参数 → 创建闭包 → 接收对象 → 解析路径 → 调用方法
