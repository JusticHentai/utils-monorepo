# includes

检查集合中是否包含指定值。

## 详细介绍

`includes` 检查值是否在集合（数组或字符串）中。使用 SameValueZero 比较。支持 fromIndex 指定起始搜索位置。

## 具体实现原理

1. **文件职责**: `includes/index.ts` 导出 `includes` 函数
2. **核心流程**: 根据 collection 类型选择数组或字符串的搜索方式
3. **关键技术点**: SameValueZero 比较、fromIndex 支持
4. **数据流向**: collection + value → 搜索 → 返回布尔值
