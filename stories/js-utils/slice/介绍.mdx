# slice

创建数组的切片。

## 详细介绍

`slice` 创建数组的切片，是 `Array.prototype.slice` 的函数式封装。

**常用场景**：
- 数组截取
- 分页取数据
- 不修改原数组的子数组获取

## 具体实现原理

1. **文件职责**: `index.ts` 封装数组 slice 操作
2. **核心流程**: 调用 Array.prototype.slice 创建子数组
3. **关键技术点**: 不修改原数组，返回新数组
4. **数据流向**: 数组 + 起止索引 → slice 操作 → 返回新子数组
