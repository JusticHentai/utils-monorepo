# pad

字符串填充工具。

## 详细介绍

`pad` 在字符串两侧填充字符至指定长度。`padStart` 仅左侧填充，`padEnd` 仅右侧填充。

**常用场景**：
- 数字格式化（如补零）
- 文本对齐
- 固定宽度字符串

## 具体实现原理

1. **文件职责**: `index.ts` 提供 pad、padStart、padEnd 三个函数
2. **核心流程**: 计算需要填充的长度 → 生成填充字符串 → 拼接到对应位置
3. **关键技术点**: pad 平分左右填充（奇数时右侧多一个），repeat 和 slice 控制填充长度
4. **数据流向**: 字符串 + 目标长度 + 填充字符 → 计算填充量 → 拼接 → 返回结果
