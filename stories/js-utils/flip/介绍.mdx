# flip

创建一个翻转接收参数顺序的函数。

## 详细介绍

`flip` 创建一个新函数，调用时将参数顺序反转后传递给原函数。在函数组合和适配器模式中很有用。

## 具体实现原理

1. **文件职责**: `flip/index.ts` 导出 `flip` 函数
2. **核心流程**: 返回新函数，将 args 数组 reverse 后传给原函数
3. **关键技术点**: `Array.prototype.reverse`、`Function.prototype.apply`
4. **数据流向**: 参数数组 → reverse → 传递给原函数
