# walk

walk 是一个用于遍历对象并处理值的异步函数，会跳过对象和数组，只处理基本类型的值。

## 使用示例

<CodeOrSourceMdx language="typescript">
  {normalDemo}
</CodeOrSourceMdx>

## 参数介绍

### obj

- 类型：`Record<string, any>`
- 描述：要遍历的对象

### cb

- 类型：`(options: { key: string; value: any; index: Record<string, any>; parent: string[] }) => any | Promise<any>`
- 描述：回调函数
  - `key`: 当前键名
  - `value`: 当前值
  - `index`: 当前对象索引
  - `parent`: 父级路径数组

### parent

- 类型：`string[]`
- 描述：父级路径数组，默认值为 `[]`

### 返回值

- 类型：`Promise<void>`
- 描述：异步遍历完成
