# replace

替换字符串中匹配的部分。

## 详细介绍

`replace` 替换字符串中匹配模式的部分，支持字符串和正则表达式模式。

**常用场景**：
- 文本搜索替换
- 模板变量替换
- 数据格式化

## 具体实现原理

1. **文件职责**: `index.ts` 封装字符串替换
2. **核心流程**: 接收字符串、模式和替换值 → 调用 String.prototype.replace
3. **关键技术点**: 支持字符串和正则表达式模式，支持替换函数
4. **数据流向**: 字符串 + 模式 + 替换 → 匹配替换 → 返回新字符串
