# getDistance

计算两点间距离的函数。

## 详细介绍

getDistance 是一个几何计算函数，核心作用是：**计算二维平面上两点之间的欧几里得距离**。

**涉及的知识点**：
- 勾股定理：`c = √(a² + b²)`
- Math.sqrt()：计算平方根
- 坐标差值：`dx = x2 - x1`，`dy = y2 - y1`

**常用场景**：
- 游戏开发：判断碰撞检测、攻击范围
- 地图应用：计算两点间直线距离
- 拖拽交互：计算拖拽距离
- 动画：基于距离计算动画时长
- 手势识别：判断滑动距离是否达标

**优势**：
- 简单直观，直接传入两个点坐标
- 返回精确的浮点数距离

## 具体实现原理

### 文件结构

- `index.ts` - 唯一源文件，包含 getDistance 函数实现

### 核心流程

1. 计算 x 坐标差值：`dx = end.x - start.x`
2. 计算 y 坐标差值：`dy = end.y - start.y`
3. 应用勾股定理：`Math.sqrt(dx * dx + dy * dy)`

### 关键技术点

- 实现：`Math.sqrt((end.x - start.x) ** 2 + (end.y - start.y) ** 2)`
- 使用 `**` 幂运算符代替 `Math.pow()`
- 结果为浮点数，可用 `Math.round()` 取整
