# pick

创建一个仅包含指定属性的对象。

## 详细介绍

`pick` 函数用于从对象中选取指定的属性创建新对象。

**与 omit 的对比**：
- `pick` - 选择要保留的属性
- `omit` - 选择要排除的属性

**常用场景**：
- 提取需要的字段
- 创建数据子集
- API 响应过滤
