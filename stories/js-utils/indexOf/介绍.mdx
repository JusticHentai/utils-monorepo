# indexOf

返回首次找到值的索引。同时提供 lastIndexOf。

## 详细介绍

`indexOf` 在数组中查找值的第一个索引。`lastIndexOf` 从右开始查找。找不到返回 -1。

### 返回值

- 类型：`number`
- 描述：索引（-1为未找到）

## 具体实现原理

1. **文件职责**: `indexOf/index.ts` 导出 indexOf（default）和 lastIndexOf（named）
2. **核心流程**: 遍历数组，SameValueZero 比较找到第一个/最后一个匹配
3. **关键技术点**: 正向/逆向遍历、fromIndex 支持负数
4. **数据流向**: 数组 → 遍历比较 → 返回匹配索引
