# get

根据路径获取对象的属性值。

## 详细介绍

`get` 支持字符串路径（`'a.b.c'`、`'a[0].b'`）和数组路径（`['a', 'b', 'c']`）。当解析值为 `undefined` 时返回指定的默认值。

## 具体实现原理
1. **文件职责**: `get/index.ts` 导出 get（default）和 toPath（named）
2. **核心流程**: 解析路径 → 逐层访问属性 → 返回值或默认值
3. **关键技术点**: 正则解析 `a[0].b` 格式路径，支持字符串和数组路径
4. **数据流向**: object + path → toPath 解析 → 逐级访问 → 返回结果
