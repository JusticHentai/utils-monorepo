# keyBy

根据迭代器返回值将集合组织成对象。

## 详细介绍

`keyBy` 根据迭代器函数的返回值或属性名作为键，将集合元素组织成一个对象。相同键的后值会覆盖前值。

**常用场景**：
- 将数组转为以 ID 为键的对象（快速查找）
- 数据索引构建
- API 响应数据结构化

### 返回值

- 类型：`Record<string, T>`
- 描述：以迭代器返回值为键的对象

## 具体实现原理

1. **文件职责**: `index.ts` 实现集合到对象的键值映射
2. **核心流程**: 遍历集合 → 对每个元素调用迭代器获取键 → 以键值对构建对象
3. **关键技术点**: 支持函数和属性名两种迭代器形式，同键覆盖策略
4. **数据流向**: 集合 + 迭代器 → 遍历生成键 → 构建映射对象 → 返回结果
