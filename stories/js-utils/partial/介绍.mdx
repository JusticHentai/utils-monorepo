# partial

创建预设部分参数的函数。

## 详细介绍

`partial` 创建一个函数，预设部分参数（从左侧）。`partialRight` 从右侧预设参数。

**常用场景**：
- 参数柯里化的替代方案
- 创建特定配置的函数变体
- 减少重复参数传递

### 返回值

- 类型：`(...args) => ReturnType<T>`
- 描述：预设部分参数的新函数

## 具体实现原理

1. **文件职责**: `index.ts` 提供 partial 和 partialRight
2. **核心流程**: 接收函数和预设参数 → 返回新函数 → 调用时合并预设和新参数
3. **关键技术点**: partial 将预设参数放前面，partialRight 放后面
4. **数据流向**: 原函数 + 预设参数 → 创建闭包 → 调用时合并参数 → 执行原函数
