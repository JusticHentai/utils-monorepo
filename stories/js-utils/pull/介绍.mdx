# pull

移除数组中的指定值（原地修改）。

## 详细介绍

`pull` 系列函数从数组中移除指定值。包括 `pullAll`、`pullAllBy`、`pullAllWith`、`pullAt`。

**常用场景**：
- 从数组中删除特定元素
- 批量移除元素
- 按条件或索引移除

### 返回值

- 类型：`T[]`
- 描述：修改后的原数组（pull/pullAll）或被移除的元素数组（pullAt）

## 具体实现原理

1. **文件职责**: `index.ts` 提供 pull、pullAll、pullAllBy、pullAllWith、pullAt 五个函数
2. **核心流程**: pull/pullAll 遍历数组移除匹配值，pullAt 按索引移除
3. **关键技术点**: 原地修改数组，splice 操作，pullAllBy/With 支持自定义比较
4. **数据流向**: 数组 + 目标值/索引 → 遍历匹配 → 移除元素 → 返回修改后的数组
