# map

遍历集合，返回每个元素经迭代器处理后的结果数组。

## 详细介绍

`map` 函数创建一个数组，其值为集合中每个元素经迭代器处理后的结果。支持数组和对象。

**常用场景**：
- 数据转换和映射
- 提取对象数组中的特定属性
- 数据格式化

## 具体实现原理

1. **文件职责**: `index.ts` 实现通用的集合映射操作
2. **核心流程**: 判断集合类型 → 遍历元素 → 调用迭代器 → 收集结果
3. **关键技术点**: 支持数组和对象两种输入类型
4. **数据流向**: 集合 + 迭代器 → 遍历元素 → 映射转换 → 返回结果数组
