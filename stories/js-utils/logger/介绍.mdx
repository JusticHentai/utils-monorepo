## 参数介绍

logger 对象包含以下方法：

### info

- 类型：`(...input: any[]) => void`
- 描述：输出 info 级别的日志，颜色为 `#7ed6df`

### debug

- 类型：`(...input: any[]) => void`
- 描述：输出 debug 级别的日志，颜色为 `#e056fd`

### warn

- 类型：`(...input: any[]) => void`
- 描述：输出 warn 级别的日志，颜色为 `#f0932b`

### error

- 类型：`(...input: any[]) => void`
- 描述：输出 error 级别的日志，颜色为 `#eb4d4b`
