# sample

从集合中随机获取元素。

## 详细介绍

`sample` 从集合中随机获取一个元素。`sampleSize` 随机获取 n 个元素。

**常用场景**：
- 随机抽样
- 随机推荐
- 测试数据生成

## 具体实现原理

1. **文件职责**: `index.ts` 提供 sample 和 sampleSize
2. **核心流程**: sample 随机索引取一个元素；sampleSize 使用 Fisher-Yates 洗牌取前 n 个
3. **关键技术点**: Math.random 生成随机索引，Fisher-Yates 算法保证均匀分布
4. **数据流向**: 集合 → 随机选取 → 返回元素
