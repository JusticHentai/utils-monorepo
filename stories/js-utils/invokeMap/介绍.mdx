# invokeMap

调用集合中每个元素的指定方法。

## 详细介绍

`invokeMap` 对集合中的每个元素调用指定方法或传入的函数，返回结果数组。

**常用场景**：
- 批量对字符串数组调用方法（如 toUpperCase）
- 对集合元素统一执行某个操作
- 函数式编程中的映射操作

### 返回值

- 类型：`R[]`
- 描述：每个元素方法调用结果组成的数组

## 具体实现原理

1. **文件职责**: `index.ts` 实现集合元素的批量方法调用
2. **核心流程**: 遍历集合 → 对每个元素调用指定方法或函数 → 收集结果
3. **关键技术点**: 区分方法名（字符串）和函数类型，动态方法调用
4. **数据流向**: 集合 + 方法/函数 → 遍历元素 → 逐个调用 → 返回结果数组
