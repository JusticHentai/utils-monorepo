# isMatch

执行部分深度比较，检查对象是否包含匹配的属性值。

## 详细介绍

`isMatch` 检查对象是否包含 source 中所有属性的匹配值。还提供 `isMatchWith`（自定义比较器）和 `matches`（创建匹配谓词）。

**常用场景**：
- 过滤符合条件的对象
- 模式匹配
- 对象属性子集验证

### 返回值

- 类型：`boolean`
- 描述：对象是否包含匹配的属性值

## 具体实现原理

1. **文件职责**: `index.ts` 提供 isMatch、isMatchWith、matches 三个函数
2. **核心流程**: 遍历 source 的属性 → 递归深度比较每个属性值
3. **关键技术点**: 部分深度比较，matches 通过闭包创建谓词函数
4. **数据流向**: 对象 + 模式 → 遍历模式属性 → 深度比较每个值 → 返回布尔值
