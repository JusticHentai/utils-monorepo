# set

根据路径设置对象的嵌套属性值。

## 详细介绍

`set` 函数用于设置对象的嵌套属性值，如果路径不存在会自动创建。

**路径格式**：
- 字符串：`'a.b.c'` 或 `'a[0].b.c'`
- 数组：`['a', '0', 'b', 'c']`

**常用场景**：
- 动态设置深层嵌套属性
- 构建复杂数据结构
- 表单数据处理
