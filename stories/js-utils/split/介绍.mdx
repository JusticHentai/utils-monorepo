# split

按分隔符拆分字符串。

## 详细介绍

`split` 按分隔符拆分字符串，是 `String.prototype.split` 的函数式封装。

**常用场景**：
- CSV 数据解析
- URL 参数解析
- 文本行分割

### 返回值

- 类型：`string[]`
- 描述：拆分后的字符串数组

## 具体实现原理

1. **文件职责**: `index.ts` 封装字符串 split
2. **核心流程**: 调用 String.prototype.split
3. **关键技术点**: 支持字符串和正则分隔符，limit 限制返回数量
4. **数据流向**: 字符串 + 分隔符 → split → 返回字符串数组
