# snakeCase

转换字符串为下划线命名格式（snake_case）。

## 详细介绍

`snakeCase` 函数将字符串转换为下划线分隔的小写格式。

**常用场景**：
- 数据库字段名转换
- API 参数命名
- Python 风格命名
