# isObject

判断值是否为 `[object Object]` 类型。

## 详细介绍

使用 `Object.prototype.toString.call(obj)` 检查值是否为纯 Object 类型。注意此实现与 lodash 不同：lodash 的 `isObject` 对数组、函数等也返回 true，而此实现仅匹配 `[object Object]` 标签。

### 返回值
- 类型：`boolean`
- 描述：如果是 [object Object] 返回 true

## 具体实现原理
1. **文件职责**: `isObject/index.ts` 导出 isObject
2. **核心流程**: `Object.prototype.toString.call(obj)` 标签检查
3. **关键技术点**: 只匹配 `[object Object]`，不匹配数组/函数等
4. **数据流向**: value → toString → 比较标签 → 返回布尔值
