# identity

返回接收到的第一个参数。

## 详细介绍

`identity` 看似无用，实际上是函数式编程中非常重要的基础工具，主要用作**默认回调/占位函数**：

- **默认 iteratee**：很多工具函数需要接收转换函数，`identity` 作为默认值表示"不做任何转换"，如 `sortBy(arr)` 不传 iteratee 时按原值排序
- **过滤假值**：`[0, 1, false, 2, '', 3].filter(identity)` => `[1, 2, 3]`
- **函数组合占位**：在 `flow`/`pipe` 中作为无操作环节，如 `flow(parse, needFormat ? format : identity, serialize)`

本质上是函数式编程的"零元素"——不改变任何东西，但让接口保持统一。
