# Timer

Timer 是一个用于计算时间差的类，可以获取本地或服务器时间。

## 开始计时示例

<CodeOrSourceMdx language="typescript">
  {startDemo}
</CodeOrSourceMdx>

## 设置时间节点示例

<CodeOrSourceMdx language="typescript">
  {setDemo}
</CodeOrSourceMdx>

## 参数介绍

### start

- 类型：`() => TimeDifference`
- 描述：开始计时，返回初始时间差对象
  - `time`: 时间戳
  - `timeDifference`: 与上一段时间差（首次为 0）
  - `timeAllDifference`: 与首次记录的时间差（首次为 0）

### set

- 类型：`() => TimeDifference`
- 描述：设置时间节点，返回当前时间差对象
  - `time`: 时间戳
  - `timeDifference`: 与上一段时间差
  - `timeAllDifference`: 与首次记录的时间差

### refresh

- 类型：`() => void`
- 描述：刷新时间状态，清空时间列表
