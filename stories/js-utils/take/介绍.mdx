# take

提取数组前 n 个元素。

## 详细介绍

`take` 提取数组前 n 个元素。还有 `takeRight`（从末尾取）、`takeWhile`/`takeRightWhile`（按条件取）。

**常用场景**：
- 获取前 N 条数据
- 分页数据截取
- 条件截取

## 具体实现原理

1. **文件职责**: `index.ts` 提供 take、takeRight、takeWhile、takeRightWhile
2. **核心流程**: take 用 slice(0, n)；takeWhile 从头遍历直到条件不满足
3. **关键技术点**: slice 截取，while 循环条件检查
4. **数据流向**: 数组 + 数量/条件 → 截取 → 返回子数组
