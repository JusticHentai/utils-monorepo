# collectError

collectError 是一个用于收集 JS 运行时错误和 Promise 错误的函数。

## 使用示例

<CodeOrSourceMdx language="typescript">
  {normalDemo}
</CodeOrSourceMdx>

## 参数介绍

### runtime

- 类型：`(errorLog: string) => any`
- 描述：运行时错误回调函数，接收错误日志字符串

### reject

- 类型：`(errorLog: string) => any`
- 描述：Promise 错误回调函数，接收错误日志字符串
