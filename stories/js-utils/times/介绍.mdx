# times

调用迭代函数 n 次。

## 详细介绍

`times` 调用迭代函数 n 次，返回每次调用结果组成的数组。

**常用场景**：
- 生成固定数量的数据
- 重复执行操作
- 创建序列

### 返回值

- 类型：`T[]`
- 描述：每次调用结果组成的数组

## 具体实现原理

1. **文件职责**: `index.ts` 实现迭代执行
2. **核心流程**: 循环 n 次 → 每次调用迭代器传入索引 → 收集结果
3. **关键技术点**: 简单循环，传入递增索引
4. **数据流向**: 次数 + 迭代器 → 循环调用 → 收集结果数组
