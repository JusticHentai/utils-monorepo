# endsWith

检查字符串是否以指定字符串结尾，同时提供 `startsWith` 检查开头。

## 详细介绍

`endsWith` 检查字符串是否以给定目标字符串结尾，可选指定检查的位置。`startsWith` 检查是否以给定目标字符串开头。两者均支持指定位置参数。

### 返回值

- 类型：`boolean`
- 描述：是否匹配

## 具体实现原理

1. **文件职责**: `endsWith/index.ts` 导出 endsWith（default）和 startsWith（named）
2. **核心流程**: 根据 position 截取字符串的相应部分，检查是否与 target 匹配
3. **关键技术点**: `String.prototype.slice` 截取、`String.prototype.indexOf` 比较
4. **数据流向**: string + position → 截取目标区域 → 与 target 比较 → 返回布尔值
