# drop

创建一个切片数组，去除数组前面或后面的 n 个元素。

## 详细介绍

`drop` 系列函数用于从数组中移除指定数量的元素，返回新数组，不修改原数组。

**包含的函数**：
- `drop` - 从数组开头移除 n 个元素
- `dropRight` - 从数组末尾移除 n 个元素
- `dropWhile` - 从开头移除元素直到断言返回 false
- `dropRightWhile` - 从末尾移除元素直到断言返回 false

**常用场景**：
- 分页数据处理
- 跳过数组前几项
- 根据条件过滤数组元素

## 具体实现原理

### 核心流程

1. **drop**: 使用 `slice(n)` 截取从索引 n 开始的元素
2. **dropRight**: 使用 `slice(0, -n)` 截取除最后 n 个元素外的部分
3. **dropWhile**: 从开头遍历，找到第一个不满足断言的索引，返回该索引后的元素
4. **dropRightWhile**: 从末尾遍历，找到第一个不满足断言的索引，返回该索引前的元素

### 关键技术点

- 使用 `Array.prototype.slice()` 进行数组切片
- 断言函数支持多种形式（函数、对象匹配、属性名）
- 返回新数组，保证不可变性
