# isWindow

isWindow 是一个用于判断对象是否是 window 的函数。

## window 示例

<CodeOrSourceMdx language="typescript">
  {windowDemo}
</CodeOrSourceMdx>

## 非 window 示例

<CodeOrSourceMdx language="typescript">
  {notWindowDemo}
</CodeOrSourceMdx>

## 参数介绍

### el

- 类型：`any`
- 描述：要检测的对象

### 返回值

- 类型：`el is Window | T`
- 描述：如果对象是 window 则返回 `true`，否则返回 `false`
