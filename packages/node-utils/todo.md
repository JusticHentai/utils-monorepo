<details>
<summary><b>📝 yargs 使用示例 - commandDir 自动加载命令</b></summary>

```typescript
// cli.ts
yargs(process.argv.slice(2))
  .commandDir('cmds', {
    extensions: ['js', 'ts'],
  })
```

`commandDir` 会扫描 `cmds` 目录，自动加载所有命令文件。每个命令文件需导出：

```typescript
// cmds/pb2ts.ts
export const command = 'pb2ts'        // 命令名
export const describe = '...'         // 命令描述  
export const builder = { ... }        // 命令参数定义
export const handler = (argv) => { }  // 命令执行逻辑
```

执行 `my-cli pb2ts` 时，会自动调用 `cmds/pb2ts.ts` 中的 `handler` 函数。

</details>
